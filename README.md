# 💳 UPI QR Code Generator (Python Project)

A simple Python-based tool that generates QR codes for UPI payments using a given UPI ID.  
It creates QR codes compatible with popular payment apps like PhonePe, Paytm, and Google Pay.

---

## 🚀 Features

- Generate UPI payment QR codes
- Supports multiple payment platforms:
  - PhonePe
  - Paytm
  - Google Pay
- Saves QR codes as image files
- Displays QR codes instantly

---

## 🛠️ Technologies Used

- Python
- qrcode library
- Pillow (PIL) for image handling

---

## 📦 Installation

Install required libraries:
pip install qrcode pillow

## ▶️ How to Run

### Step 1: Clone the repository
git clone https://github.com/your-username/UPI-QR-Code-Generator-Python.git

### Step 2: Navigate to project folder
cd UPI-QR-Code-Generator-Python

### Step 3: Run the script
python main.py

## 🧾 How it Works
1. User enters UPI ID
2. Program generates UPI payment links
3. QR codes are created for:
a.PhonePe
b.Paytm
c.Google Pay
4. QR images are saved and displayed

## 📌 Example UPI Format Used
upi://pay?pa=UPI_ID&pn=Recipient Name&am=Amount&cu=INR
